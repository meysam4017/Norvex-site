RTK GNSS Systems

High-Precision RTK GNSS Systems

Norvex RTK systems provide centimeter-level positioning accuracy through real-time GNSS correction technology, enabling reliable performance in demanding industrial and mission-critical environments.

What is RTK Positioning?

Real-Time Kinematic (RTK) is a GNSS positioning technique that improves accuracy from meters to centimeters by applying real-time correction data. A complete RTK system consists of a reference station (CORS), a rover receiver, and a correction data link that ensures precise positioning in the field.

RTK System Components

01 — RTK ROVER

Field Receivers for Centimeter Accuracy

RTK rovers operate in the field, receiving correction data and delivering precise positioning for surveying and industrial operations.

  • Centimeter-level accuracy
  • Real-time positioning
  • Reliable field performance
02 — CORS STATION

Reference Stations for RTK Networks

Continuously Operating Reference Stations (CORS) provide high-precision correction data and form the foundation of RTK positioning systems.

  • Stable and continuous GNSS observation
  • High-quality correction generation
  • Network-based scalability
03 — CORRECTION SOFTWARE

RTK Correction Distribution Systems

Software platforms deliver GNSS correction data via NTRIP and communication networks, ensuring real-time positioning performance across systems.

  • NTRIP-based correction delivery
  • Scalable system architecture
  • Reliable data communication
