What is RTK GNSS?

RTK, or Real-Time Kinematic, is a high-precision GNSS positioning method that improves accuracy from meter-level to centimeter-level by using real-time correction data. Unlike standard GNSS positioning, RTK compares satellite observations from a fixed reference station and a moving rover receiver to reduce errors caused by atmospheric effects, satellite orbit uncertainty, and receiver clock bias.

RTK GNSS is widely used when accurate and repeatable positioning is required, including land surveying, construction layout, machine guidance, precision agriculture, and autonomous navigation.

How RTK Positioning Works

A complete RTK system typically includes a reference station or CORS station, a rover receiver in the field, and a communication link that transfers correction data in real time. The reference station observes the same GNSS satellites as the rover and computes correction information, which is then transmitted through radio, internet, or NTRIP communication. The rover applies these corrections instantly to achieve centimeter-level accuracy.

  • GNSS satellites provide raw positioning signals
  • A reference station calculates correction data
  • Corrections are transmitted to the rover in real time
  • The rover resolves precise carrier-phase positioning

Main Components of an RTK GNSS System

01 — RTK ROVER

Rover Receivers for Field Positioning

The rover is the mobile RTK GNSS receiver used in the field. It receives satellite signals and real-time correction data to produce accurate coordinates for surveying, construction, and industrial applications.

  • Centimeter-level real-time positioning
  • Reliable field performance
  • Designed for demanding outdoor environments
02 — CORS STATION

Reference Stations and CORS Networks

A CORS station, or Continuously Operating Reference Station, serves as the fixed reference point in RTK positioning. It continuously tracks GNSS signals and generates the correction data needed by rover receivers.

  • Stable and continuous GNSS observation
  • Correction generation for RTK positioning
  • Scalable from local base station to network RTK
03 — CORRECTION SOFTWARE

Correction Data Delivery and NTRIP Systems

RTK correction software distributes real-time correction data from reference stations to users. In modern deployments, this is often done through NTRIP services over cellular or internet networks.

  • NTRIP-based correction delivery
  • Scalable architecture for multiple users
  • Reliable communication for real-time operation

RTK Accuracy and Key Benefits

The main advantage of RTK GNSS is its ability to deliver centimeter-level positioning in real time. This makes RTK one of the most effective technologies for applications that require precise alignment, repeatability, and navigation confidence.

  • Centimeter-level positioning accuracy
  • Real-time field operation
  • Improved repeatability and productivity
  • Scalable from local projects to wide-area correction networks
  • Suitable for integration with GNSS + INS architectures
