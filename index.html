RTK GNSS

Centimeter-Level Positioning for Professional Applications

RTK GNSS improves positioning accuracy from meter-level to centimeter-level by using correction data from reference stations, CORS networks, or Network RTK services. It is widely used in surveying, construction, machine guidance, precision agriculture, mining, and autonomous systems.

  • RTK rover and CORS reference station workflows
  • NTRIP and cellular correction delivery
  • Centimeter-level positioning for field operations
